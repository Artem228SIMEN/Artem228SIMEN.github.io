Объекты на карте

Интерактивная карта "Экстренные службы города Новокузнецк"

Описание проекта

Создано с использованием библиотек Bootstrap, Leaflet.

Особенности
  • Полноэкранный удобный интерфейс с навигационной панелью адаптированный для мобильных устройств.
  • Подгрузка через jQuery внешние GeoJSON файлы
  • Многослойная кластеризация маркеров с помощью leaflet плагина
  • Элегантный поиск с использованием автозаполнения typeahead.js
  • Адаптивная боковая панель, которая включает сортировку и фильтрацию с помощью list.js
  • Значки маркеров, включенные в управление сгруппированным слоем с помощью grouped layer control plugin
